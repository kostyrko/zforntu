HTML i CSS: Obrazy i Sprite

Opublikowano pon 25 maja 2020 w css • 1 min read

Semantyczny HTML

Znaczeniowy kod HTML - wykorzystanie tagów HTML zgodnie z przeznaczeniem

Wstawianie obrazu

Z perspektyw css/background wtedy kiedy zdjęcie nie jest treścią - kiedy jest zaledwie lukrem

Kiedy zdjęcie jest integralną częścią (np. wykres) wówczas jest istotne by znalazło się w img. tak aby czytnik stron, mógł przeczytać treść zawartą w alt

PNG - bezstratna kompresja stosowany do ikon i logotypów, gdzie rozmycie pikseli i nie jest dopuszczalne

JPG - do zdjęć, kompresja stratna - gdzie ew. rozmycie jest dopuszczalne

CSS Sprite - tworzenie wielu obrazów w jednym pliku PNG, tak aby wychodziło jedno zapytanie do serwera a nie wiele dla każdego obrazu (oszczędza czas) - korzystanie z takiego podejścia wymaga ustawienia tła aby wyświetlało jeden obraz (będący fragmentem dużego obrazu) w danym miejscu. Z tego podejścia korzysta się w trakcie tworzenia animacji

srcset - pozwala na podmianę obrazu wy przypadku innej gęstości pikseli

na jeszcze więcej pozwala tag (warunki sprawdzane są od góry)

<picture>
  <source media="(min-width: 45em)" srcset="large.jpg">
  <source media="(min-width: 32em)" srcset="med.jpg">
  <img src="small.jpg" alt="The president giving an award.">
</picture>
